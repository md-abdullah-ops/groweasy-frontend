# GrowEasy AI CSV Importer - Frontend

This is the frontend application for the GrowEasy AI-powered CSV Importer assignment. It provides a responsive, modern UI to upload unstructured CSV files, preview the data, and display the final CRM records extracted by the AI backend.

## Tech Stack
* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **CSV Parsing:** PapaParse (for client-side preview)
* **Icons:** Lucide React
* **Deployment:** Vercel

## Features
* **Modern UI:** Clean, responsive layout with loading states and error handling.
* **Client-Side Preview:** Instantly parses and displays the first 5 rows of the uploaded CSV before sending data to the server.
* **AI Results Table:** Displays the successfully imported records and counts skipped records based on the AI's mapping.

## Setup Instructions

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   git clone [Paste your frontend repo URL here]

2. **Navigate to the directory:**
   cd [Your frontend folder name]

3. **Install dependencies:**
   npm install

4. **Start the development server:**
   npm run dev

5. **Open the application:**
   Open http://localhost:3000 in your browser.

*Note: Ensure the Node.js backend is running concurrently to handle the file upload and AI extraction.*
